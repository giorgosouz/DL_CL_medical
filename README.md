# DL_CL_medical

Course project on contrastive learning for medical image segmentation.

## Repository contents
- `dataloaders.py`: dataset loading utilities
- `models.py`: model definitions
- `losses.py`: loss functions
- `layers_bn.py`: model building blocks
- `f1_utils.py`: evaluation helpers
- `utils.py`: shared utilities
- `tr_baseline.py`: baseline training script
- `experiment_init/`, `train_model/`, `scripts/`: experiment helpers and training scripts
- `Deep_Learning_Project.pdf`: project report

## Notes
- The repository combines code and report material from a course project.
- Reading the report alongside the training scripts gives the clearest view of the intended workflow.
